# lilRobots

robots.txt and llms.txt Checker

Check any site's robots.txt and llms.txt to see who is allowed to crawl it, which bots are blocked, and whether AI crawlers are addressed.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
